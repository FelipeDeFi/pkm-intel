☁ Google Drive Sync
Salva em uma pasta visivel do Google Drive
Como configurar (5 minutos, gratuito): 1. Acesse console.cloud.google.com
2. Crie um projeto (ex: PKM Intel)
3. Vá em APIs & Services → Enable APIs → habilite Google Drive API
4. Vá em APIs & Services → Credentials → Create Credentials → OAuth client ID
5. Tipo: Web application
6. Em Authorized JavaScript origins, adicione: null (para arquivo local)
7. Copie o Client ID gerado e cole abaixo
Client ID OAuth 2.0
⚠️ O Client ID fica salvo apenas no seu navegador (localStorage). Nenhum dado é enviado para servidores externos além do seu próprio Google Drive.
